# JointAtlas — Complete Cartilage & Sciatica-Friendly Movement Plan

A polished static website for a complete non-surgical, cartilage-friendly movement plan. It is designed for a late-40s overweight patient and covers knee, hip, ankle/foot, and low-back/sciatica-type leg pain patterns.

## Files

- `index.html` — main website
- `styles.css` — full responsive UI system, print styles, premium visual design
- `script.js` — exercise library, filters, workout checklist, phase tabs, decision system
- `sources.md` — source notes and demo links

## How to use

Open `index.html` in a browser. No build step is required.

## Deploy

Upload this folder to any static host, including Netlify, Vercel, GitHub Pages, Cloudflare Pages, or a normal web server.

## Medical note

The content is educational and does not replace a doctor, physical therapist, or emergency care. The website includes red flags for when to stop and seek help.
